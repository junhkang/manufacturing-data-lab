# 🔬 실습 시나리오: 열화상 기반 비파괴 검사 자동화

## 🏭 배경

- 자동차 부품(예: 사이드 몰딩)은 가스 사출 성형 방식으로 제작됨
    
- 내부 공기량이나 금형 조건에 따라 **두께 불균일**이 자주 발생
    
- 현재는 **파괴 검사**로만 불량 여부 확인 가능 → 시간과 비용 손실
    

## ❌ 기존 문제 상황

- 불량 여부는 생산 종료 후에야 확인 가능 (검사 시점 지연)
    
- 불량 발견 시 앞뒤 제품 수십 개 재검사 또는 폐기
    
- 육안 검사 + 샘플 검사 → 전수 검사 불가 → 누락 위험
    

## ✅ 해결 방안

- **열화상 센서를 통해 수집한 표면 온도 데이터**로 품질 상태를 추정
    
- 품질 기준(두께)을 바탕으로 불량 여부 자동 판별
    
- 머신러닝(SVM)을 통해 **자동 진단 모델** 구성
    

---

## 🎯 실습 목표

- 열화상 벡터 데이터를 시각화하고 구조를 이해한다.
    
- 관심 영역을 추출하고, 기준 기반 라벨을 생성한다.
    
- 단순 진단 로직 및 SVM 분류 모델을 구성한다.
    
- 오분류 결과를 해석하고 리포트로 정리한다.
    

---

## 🧪 실습 단계 요약

### 1. 데이터 구조 파악 및 시각화

- `left_data.csv`: 열화상 센서 벡터 (예: 256x320)
    
- 1개의 벡터를 이미지로 시각화 (`reshape` + `imshow`)
    
- 평균 온도, 히트맵 등 시각적으로 확인
    

### 2. 관심 영역 추출

- 전체 이미지에서 ROI 또는 중심라인 벡터 추출
    
- Skeleton 추출 또는 평균 열값 라인으로 축소 (예: 1x80)
    
- 정규화 (normalize) 수행
    

### 3. 기준 기반 라벨 생성

- `left_label.json`의 두께 정보 활용
    
- 기준: `0.8mm < thickness < 1.5mm` → 양품 / 그 외 → 불량
    
- 제품별 라벨 컬럼 생성
    

### 4. 품질 진단 자동화

- 전처리 + 기준 비교를 함수로 구성
    
- 각 샘플에 대해 `OK / NG` 판단
    
- 전체 샘플에서 품질 진단 요약
    

### 5. SVM 분류 모델 학습

- 입력: 관심영역 벡터 (1x80)
    
- 라벨: 양품 / 불량 (binary)
    
- 학습 후 정확도 평가, 혼동 행렬 확인
    

### 6. 오분류 시각화 및 리뷰

- 틀리게 예측된 샘플 이미지 추출
    
- 온도 분포 시각화 → 오류 원인 탐색
    
- 기준에 가까운 경계 사례 분석
    

### 7. 결과 요약 및 리포트 생성

- 정확도, 재현율 등 핵심 지표 계산
    
- 전체 품질 상태 통계 요약
    
- PDF 또는 CSV 형식 리포트 출력 (선택 사항)
